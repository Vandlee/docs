# U-Ui System

U-Ui System es un conjunto de utilidades CSS para ayudarle a crear diseños personalizados de forma más eficiente. Permite maquetar rápidamente diseños personalizados.

## Instalación

Instala el paquete en el directorio de tu proyecto con:

<!-- #default-branch-switch -->

```bash
npm install @u_ui/system @emotion/react @emotion/styled
```
